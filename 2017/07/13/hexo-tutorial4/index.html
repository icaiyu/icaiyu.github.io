我们仔细看了看，感觉这个博客网页外观还行，但是，还不是很合心意。也就是说，我们对这个外观主题不感兴趣，这里的主题的意思是类似QQ皮肤一样的东西，我们的博客网页也有皮肤，可以随意更换，不需要充钱，我们高端一点，称之为主题。
理论上，我们可以自己打造属于我们的独一无二的主题，但是，如果你在看我的教程，十有八九，你还不懂如何打造主题（没啥好羞愧的，术业有专攻嘛）
于是，我们可以使用别人打造好的主题。
hexo的博客有哪些主题呢，大家可以到这个网页上查看
https://hexo.io/themes/
里面真的是款式众多，而且都非常漂亮，有高端奢华，有低调简洁，各种风格的设计，总有满足你的款式。
挑选我们喜欢的款式，就拿我的博客来说吧，我喜欢这个简洁大方的风格，这套主题的名字叫next。找到之后点开示例，你就可以看到这套主题长啥样。这些可不是仅仅是示例博客，而是真正的个人博客或者团队博客，是这套主题的开发者的博客，大家有空多去膜拜一下，那些都是真正的大神。我们进入这些博客之后找到GITHUB关键字眼，

这一般是一个链接，点击跳转过去，一般就是开发者的代码仓库，大神说不定有很多项目代码，会有很多仓库，我们找找，就很快能找到主题所在的仓库

点击进去，找到右边绿色下载的按钮，

点击右边的Download ZIP下载到我们的电脑。我们就把主题的代码下好了，解压，然后把文件夹复制到myBlog文件夹里面的theme文件夹里面中，与一个叫landscape的文件夹放到一起。这个landscape文件夹就是我们默认的主题文件夹，我们刚刚看自己的博客网页的时候，是不是见到最上面有一张横幅图片，上面就是一个地平线，太阳正在升起。所以这就是我们的默认主题名字叫landscape的原因了。我们把新下载的主题文件夹放到旁边，表示这个文件夹与landscape地位相当，都是主题文件夹，我们的博客可以使用默认主题，也可以切换成我们新下载的主题，我们把这个文件夹的名字修改一下，免得太长难记，我改成next，这是该主题的标准名字。值得注意的是，我们之前解压的时候，有可能会出现文件夹再套一个文件夹的情况，我们要确保，我们一打开next文件夹就看到一堆文件，如下图，而不是又出现一个和原来文件夹名字相同的文件夹。

文件虽然多，其中需要关注的，其实就一个，那就是_config.yml文件，和我们之前提到的站点配置文件名字一样，不过，我们称这个为主题配置文件。
这个主题配置文件能配置的东西可比我们之前见到的站点配置文件多了，凡是涉及到博客网页的外观颜值的都是在这里设置的，比如字体，比如背景图片，比如各种logo，还有很多比如博客的评论功能，打赏功能这些也都是主题配置文件设置的，关于这套next主题的配置，有很多好玩的东西，你可以到网上搜索一下，看看网友们是如何配制出狂拽酷炸的next主题的。这里就不多说了。
值得一提的是，这个next主题还提供了三种不同的风格，默认的是Muse风格，如果你想使用和我的博客一样的风格，你可以在主题配置文件80多行那里，找到
schema 关键字眼。
我们在scheme: Muse前面添加一个井号把这个配置注释掉，使配置无效
然后把下一行的scheme: Mist前面的井号删除，使这个配置有效。
这时，你运行
hexo s
开启服务器
到浏览器上看看我们的网页，你就会发现啥也没变，因为我们还没把主题从默认的landscape换成next。所以先按ctrl+C停止服务器的运行，我们打开站点配置文件！像选择主题这种大事肯定是站点配置文件负责的。
在底下看到
theme: landscape
关键字眼，我们把landscape删除，换上next（你要确保之前有把next主题文件夹修改文件名成next，总之，你得输入myBlog/theme文件夹里面所包含的文件夹的名字，里面默认有landscape，还有我们刚刚复制进去的next主题文件夹）
还有值得注意的是删除landscape不要用力过猛，把theme: 后面的空格也删除了。
正确的配置应该是
theme: next
冒号后面有一个空格！！！！！！！！！
其他所有的配置都是一样的，冒号后面要有一个空格！！！！！

这时候，我们运行
hexo g
它会自动删除旧的public文件夹生成新的public
我们再运行
hexo s
这时候就能够看到全新的主题了！！！
实际上，由于next主题实在是太受欢迎了，于是专门有了一个官方网站来告诉你如何使用这套主题。
http://theme-next.iissnan.com/
这也是我们中国人设计的主题，因此，网站你是看的懂的。
